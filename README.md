# hapi-axios
A Hapi plugin for axios

# install
```
npm install github:dhso/hapi-axios
```

# axios github

```
https://github.com/axios/axios
```

# options
```
//set baseUrl
baseURL: 'http://localhost:8081/api'

//set timeout
timeout: 8000
```
